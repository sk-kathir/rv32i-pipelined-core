# RV32I Pipelined Core

A planned 5-stage pipelined RV32I processor implemented in SystemVerilog.

## Status

Currently learning digital logic, Verilog, and computer architecture before beginning implementation.

## Goal

To design and simulate a 32-bit RV32I processor with a 5-stage pipeline:

- Instruction Fetch
- Instruction Decode
- Execute
- Memory Access
- Write Back

## Tools

- SystemVerilog
- Icarus Verilog
- GTKWave
- Yosys
- netlistsvg
